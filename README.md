# Learn Git
Tutorial to learn github. 

__Git Basics__
```bash
git clone https://github.com/epccVV/LearnGit
cd LearnGit
echo -e "my comment" > file.txt
git add . 
git commit -m "my comment"
git push origin master
```

#### Get Git Status 
```bash
git status
git status -s
git status -v
``` 

#### Create a development branch 
```bash
git checkout -b development 
```
